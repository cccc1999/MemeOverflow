== Getting Started

1. Fork the repo

2. build your own feature on your own branch. i.e git branch <branchname>

3. write some test. 1-10 test is required per each feature. you decide before you ask us to pull, we decide if your feature is dangerous or not.

4. have a sense of humor and realize that you just contributed to a open source project high fives! credits to you!
